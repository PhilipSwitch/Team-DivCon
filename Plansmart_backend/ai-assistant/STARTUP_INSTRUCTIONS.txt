========== TROUBLESHOOTING ==========

"Failed to fetch" Error:
- Make sure BOTH backend (port 5000) and frontend (port 3000) are running
- Check that no other application is using these ports
- Check your firewall settings

Backend not starting:
- Make sure Python is installed
- Run: pip install -r requirements.txt
- Check that you're in the correct backend directory

Frontend not loading:
- Make sure the HTTP server started successfully
- Try a different port if 3000 is already in use

Connection refused:
- Make sure both servers are running (check both command prompts)
- Wait a few seconds after starting for servers to fully initialize
